> 로그인, 회원가입 화면
> > 1. 앱 접속 시 보이게 되는 최초 화면으로, 로그인고 회원가입을 할 수 있음.

### 마이 페이지
![마이페이지](https://user-images.githubusercontent.com/67571328/161932072-f203c04b-2a65-4637-8843-1fa77f8629c5.jpeg)
> 마이 페이지 / 설정 페이지
> > 1. 유저의 정보를 확인하고 수정할 수 있는 화면.
